# Airmon-ng

\# Monitor Mode

Copy

```
sudo airmon-ng start wlan0 11 # Specific Channel
```

\# Check Interfering Processes

Copy

```
sudo airmon-ng check
```

\# Kill Interfering Processes (If encountering problems)

Copy

```
sudo airmon-ng check kill
```
